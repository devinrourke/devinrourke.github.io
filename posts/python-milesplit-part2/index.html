milesplit, part 2 - plotting

Posted on Thu 29 December 2016 in Projects

As a High School Cross Country coach, I visit co.milesplit.com quite often. So, for a fun weekendlong project, I thought I'd try to visualize some runner data.

This is Part 2 - Plotting. Head over to Part 1 of this series of posts if you want to see how I got all the data. Additionally, if you'd rather just see the whole jupyter notebook file on nbviewer, click here.

I gave a "Tuesday Nerd Talk" about this project! Check it out here

The completeness of the milesplit database is somewhat lacking:

years = ['2000','2001','2002','2003','2004','2005','2006','2007',
         '2008','2009','2010','2011','2012','2013','2014','2015','2016']
years = map(int, years)

ind = 0
for key in sorted(df.Time.groupby(df.State).groups.keys()):    
    ind = ind + 1
    ax = plt.subplot(6, 10, ind)
    plt.axis([2000, 2016, 0, 1000])
    plt.xticks([])
    frame1 = plt.gca()
    frame1.axes.get_xaxis().set_visible(False)
    df.Time.groupby([df.State, df.Year]).count().get(key).reindex(index=years).plot.bar(
        title = key, figsize = (16, 9), fontsize = 12, sharey = True, sharex = False) 
    plt.tight_layout()
    plt.xlabel('')
plt.show()
